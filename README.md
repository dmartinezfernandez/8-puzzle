# 8-puzzle

8-puzzle game for kids

## Getting started

Vue 3 from scratch with no features:

```
vue create eight-puzzle
vue add vue-next
```

[Using Vue in Visual Studio Code](https://code.visualstudio.com/docs/nodejs/vuejs-tutorial)
